# Wayfinders

In times of uncertainty, whether traveling plains, mountains, tunnels, forests, or high seas, a wayfinder knew how to plot a course and get people from one place to another.

This package contains a number of simple algorithms implemented in Python for school projects and actually useful programs for people to use at their leisure.

## Search Agents

School projects to implement simple search algorithms in Python.

## Map Boost

A set of desktop and mobile applications for arranging Google Maps location and datetime data for easy parsing in spreadsheets.
